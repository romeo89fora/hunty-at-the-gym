# Hunty at the Gym – PWA (No Build, No Framework)

- Single-file PWA. No external libraries.
- Works over HTTPS. Add to Home Screen on iPhone.
- Data is saved in localStorage.

## Deploy on Hostinger
1) Create folder `/public_html/hunty`.
2) Upload all files here.
3) Visit `https://YOURDOMAIN/hunty/` in Safari on iPhone.
4) Share → Add to Home Screen.

## Files
- index.html
- manifest.webmanifest
- sw.js
- /icons/*.png
